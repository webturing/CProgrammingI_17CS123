##  二维数组的定义：
-  指定行列,int a[3][4]
-  指定列（需要初始化器）
```
int a[][3]={{1,2,},{3,4,5},{0}}
int a[][3]={1,2,3,4,5};
```
##  二维数组（ROW*COL）
```
int a[ROW][COL];//a是二维数组，a[x]是一维数组,a[x][y]是int
// 逻辑结构 ROW行 COL列（下标从0开始）
//  物理（内存）结构 是名为a[0]长度为ROW*COL的一维数组
```
-  二维=>一维公式：
```a[i][j]=a[0][i*COL+j]```
-  一维=>二维公式：
```a[0][x]=a[x/COL][x%COL]```
